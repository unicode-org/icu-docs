CAPI const char* ucol_getAvailable (int32_t index)

Get a locale for which collation rules are available

Documentation

Get a locale for which collation rules are available. A UCollator in a locale returned by this function will perform the correct collation for the locale.
Returns:
A locale for which collation rules are available, or 0 if none.
Parameters:
index - The index of the desired locale.
