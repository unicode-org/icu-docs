Downloading ICU

Official Releases

If you want to use ICU (as opposed to developing it), it is recommended that you download an official packaged version of the ICU source code. These versions are tested more thoroughly than day-to-day development builds of the system, and they are packaged in zip and tar files for convenient download. Here are several recent releases of ICU that are available:

Release ICU4C ICU4J ICU4JNI Major Changes
ICU 3.6 3.6 3.6 3.6 Unicode 5.0 update, UCA 5.0 update, CLDR 1.4 update, ICU4C charset detection, improved ICU4C break iterator
ICU 3.4 3.4.1 3.4.5 3.4 Unicode 4.1 update, CLDR 1.3 update, ICU4J charset detection, ICU4C high performance text access
ICU 3.2 3.2.1 3.2 3.2 CLDR 1.2 integration.
Old ICU       These are the previous releases of ICU. Due to the age of these older releases, it is recommended that you use a newer version of ICU for current development.

For more details on building ICU once you've checked out the code, please see the readme.html in ICU.
