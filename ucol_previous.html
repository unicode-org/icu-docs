CAPI int32_t ucol_previous ( UCollationElements *elems, UErrorCode *status)

Get the ordering priority of the previous collation element in the text

Documentation

Get the ordering priority of the previous collation element in the text. A single character may contain more than one collation element.
Returns:
The previous collation elements ordering, or UCOL_NULLORDER if the end of the text is reached.
Parameters:
elems - The UCollationElements containing the text.
status - A pointer to an UErrorCode to receive any errors.
