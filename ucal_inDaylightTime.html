CAPI bool_t ucal_inDaylightTime ( const UCalendar* cal, UErrorCode* status )

Determine if a UCalendar is currently in daylight savings time

Documentation

Determine if a UCalendar is currently in daylight savings time. Daylight savings time is not used in all parts of the world.
Returns:
TRUE if cal is currently in daylight savings time, FALSE otherwise
Parameters:
cal - The UCalendar to query.
status - A pointer to an UErrorCode to receive any errors
