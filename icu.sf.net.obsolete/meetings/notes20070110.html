ICU 3.8 Planning

A modalities of planning ICU 3.8 release with features and bug fixes were discussed.

Key Points:
  • IBM proposed to ICU 3.8 to be released in second half of September, 2007 (tentative). The release date is acceptable to Google, Apple and Yahoo
  • Deborah expressed concern that some key improvements to conversion API proposed by Apple may not be ready in time for 3.8
  • Google would like quarterly stable snap shots or milestone releases
  • Milestone releases should be versioned with odd numbers and reference releases should be versioned with even numbers.
                e.g.:
Reference release     - 3.6.0
Milestone 1              - 3.7.1
Milestone 2              - 3.7.2
...
Reference Release    - 3.8.0
Maintenance Release  - 3.8.1
