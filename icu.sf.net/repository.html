A bit of history:  The ICU project moved to using a Subversion source code repository and a Trac bug database on Nov 30, 2006.  These replace our original CVS source code repository and Jitterbug bug data base.  All history from the older systems has been migrated into the new, so there should normally be no need to refer back to Jitterbug or CVS.
