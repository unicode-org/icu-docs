Plugin for Eclipse 3.3 

Yoshito gave a status of ICU4J plugin in Eclipse 3.3
  • ICU plugin does not export all packages, e.g.: com.ibm.icu.impl. This does not align with the overall strategy of Eclipse. So they will export all packages
  • The manifest file for Eclipse plugin does not have the correct version
  • Some consumers are complaining about classes being disabled. Eclipse contact suggested providing a fragment to override the whole class.This fragment can depend on JDK 1.4
  • 3.6 will be adopted by Eclipse 3.3
