U_CAPI void udat_set2DigitYearStart ( UDateFormat *fmt, UDate d, UErrorCode *status)

Set the year relative to which all 2-digit years will be interpreted

Documentation

Set the year relative to which all 2-digit years will be interpreted. For example, if the 2-digit start year is 2100, the year 99 will be interpreted as 2199.
Parameters:
fmt - The formatter to set.
d - The year relative to which all 2-digit years will be interpreted.
status - A pointer to an UErrorCode to receive any errors
