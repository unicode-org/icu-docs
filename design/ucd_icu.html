Unicode Character Database properties in ICU APIs

The following table shows all Unicode Character Database properties (except for purely "extracted" ones and Unihan) and the corresponding ICU APIs. Most of the time, ICU4C provides functions in uchar.h and ICU4J provides parallel functions in the UCharacter class.

N/I indicates if a property is normative or informative.

See the Unicode Character Database itself for comparison.

Function names with a * are proposed, with names according to PropertyAliases.txt.
Unless otherwise noted, C names u_charXYZ correspond to Java names UCharacter.getXYZ and u_isBCD correspond to UCharacter.isBCD.
New APIs for binary properties are proposed as [u_]isU<PropertyNameWithoutUnderlines>.
Where there are no functions available nor proposed for a property, none is noted in the function column.
