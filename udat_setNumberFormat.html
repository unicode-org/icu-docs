CAPI void udat_setNumberFormat ( UDateFormat* fmt, const UNumberFormat* numberFormatToSet)

Set the UNumberFormat associated with an UDateFormat

Documentation

Set the UNumberFormat associated with an UDateFormat. A UDateFormat uses a UNumberFormat to format numbers within a date, for example the day number.
Parameters:
fmt - The formatter to set.
numberFormatToSet - A pointer to the UNumberFormat to be used by fmt to format numbers.
