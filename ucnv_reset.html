CAPI void U_EXPORT2 ucnv_reset (UConverter * converter)

Resets the state of stateful conversion to the default state

Documentation

Resets the state of stateful conversion to the default state. This is used in the case of error to restart a conversion from a known default state. it will also empty the internal output buffers.
Parameters:
converter - the Unicode converter
