Submitting ICU Bugs and Feature Requests

If you would like to submit a bug report, submit a patch for a bug, submit an RFE (Request For Enhancement) or search the ICU bug database, please visit http://bugs.icu-project.org/trac/.

If you are encountering a build problem, encountering an API usage problem, or you have a question about ICU, you will usually get a faster response from one of our mailing lists.

Excel Spreadsheet View

An Excel spreadsheet view of the existing Trac Tickets can be found in traclist.xls. Data Menu→Refresh Data will refresh from the Trac database while in Excel.
