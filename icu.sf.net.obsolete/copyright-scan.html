What is the copyright scanner?

ICU requires all files checked in to have a current copyright with the current year. Usually this is in a comment at the top of the file. The scanner is looking for "Copyright" and the current year (such as 2005) on the same line, within the first few lines of file data.

Exclusion list

If a file can not contain a copyright (such as a binary file), simply add the file's path to the file icu/source/tools/cpysearch/cpyskip.txt, and check in cpyskip.txt. After checking in cpyskip.txt, you will be able to check in your new file.

Complete details on this subject can be found in the ICU Wiki page.
