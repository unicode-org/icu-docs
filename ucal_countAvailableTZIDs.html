U_CAPI int32_t ucal_countAvailableTZIDs (int32_t rawOffset)

Determine how many TimeZones exist with a certain offset

Documentation

Determine how many TimeZones exist with a certain offset. This function is most useful as determining the loop ending condition for calls to ucal_getAvailableTZIDs.
Returns:
The number of TimeZones with rawOffset.
Parameters:
rawOffset - The desired GMT offset.
